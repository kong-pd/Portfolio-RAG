## 2. 量化边界（所有数字的唯一出处）

| 项 | 值 | 备注 |
|---|---|---|
| 单文件大小 | ≤ 20 MB | 前端拦截 + 后端 413 兜底 |
| 支持格式 | PDF / TXT / MD | docx 等一律 415 |
| 分块 | 512 token，重叠 64 | 评测驱动调参，见《05》决策表 |
| 向量维度 | 1,536 | text-embedding-3-small |
| 检索 | k = 5，余弦阈值 0.75 | 低于阈值 → 拒答（《07》F-04） |
| 多轮上下文 | 最近 6 轮 | 更早轮次不注入 |
| Token 有效期 | Access 15 min / Refresh 7 d | Refresh 为不透明 UUID，可吊销 |
| 密码 | ≥ 8 位 | BCrypt 存储 |
| 问题长度 | ≤ 2,000 字 | 契约新增（《06》） |
| 并发 | 单实例 ≥ 20 用户 | 嵌入任务异步化 |
| 向量规模 | < 100 万（IVFFlat lists=100） | 超出 → 迁移 HNSW |
| 性能 | 首 token ≤ 3 s；≤5 MB 文档处理 ≤ 10 s | NFR |

## 3. 范围外（Out of Scope）及理由

| 不做 | 一句话理由 |
|---|---|
| 原生移动端 App | Web 响应式（1280×720 主场景）已覆盖演示需求 |
| 实时协同文档编辑 | 与问答核心无关，复杂度不成比例 |
| 扫描件 / 图片 PDF 的 OCR | 引入额外管线与质量不确定性，留作 v2 |
| 模型微调或训练 | 项目主张正是「无需训练、RAG 即用」 |
| 第三方云存储集成（Google Drive 等） | OAuth 集成成本高，本地上传已闭环 |
| 企业 SSO（SAML / OIDC） | JWT 已满足多用户演示，SSO 是企业版命题 |
